Introduction

This documentation covers JavaScript fundamentals and advanced concepts including closures, promises, and asynchronous behavior.

Data Types

JavaScript has primitive and non-primitive data types.

  • Primitive: string, number, boolean
  • Non-Primitive: object, array

Variables

JavaScript has var, let, and const.

let a = 10; const b = 20;

Arrays

Used to store multiple values.

let arr = [1,2,3]; arr.map(x => x * 2);

Objects

Objects store key-value pairs.

let user = {name: "Mukul", age: 21};

Functions

Functions are reusable blocks of code.

const add = (a,b) => a + b;

Callback Functions

A callback is a function passed into another function.

function greet(name, callback) { console.log(name); callback(); }

Closures

Closures allow a function to access variables from outer scope.

function outer(){ let count = 0; return function(){ count++; } }

Hoisting

Variables and functions are moved to top during execution.

Promises

Promises handle asynchronous operations.

new Promise((res,rej)=>{});

setTimeout

Used to delay execution.

setTimeout(()=>console.log("Hi"),1000);
